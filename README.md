<details>
<summary><strong>⬡ &nbsp;PhishGuard — AI-Driven Phishing URL Detection Engine</strong></summary>
<br/>

A browser-integrated phishing detection system that uses natural language processing and feature engineering to classify URLs and webpage content as benign or malicious in real time — protecting users before page load.

<div align="center">

| Attribute | Detail |
|:---|:---|
| **Stack** | Python · XGBoost · BERT · Flask · JavaScript · Chrome Extension API · SQLite |
| **Scale** | 1M+ URL dataset for training · real-time classification in <200ms |
| **Performance** | 98.1% accuracy · AUC-ROC 0.99 · <0.8% false positive rate |
| **Security** | No URL data stored externally · on-device inference · HTTPS-only API |
| **Impact** | Detected 94% of zero-day phishing URLs not flagged by Google Safe Browsing |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-6d28d9?style=flat-square&logo=github&logoColor=white)](https://github.com/GopinathP7/phishguard) |

</div>

PhishGuard extracts 35 URL-level, domain-level, and lexical features combined with a fine-tuned BERT classifier for content-level semantic analysis. The Chrome extension intercepts navigation events and queries the local Flask API, rendering a threat score overlay before the user reaches a potentially malicious page.

</details>

<details>
<summary><strong>⬡ &nbsp;CryptoVault — Secure File Encryption & Steganography Tool</strong></summary>
<br/>

A command-line and GUI hybrid tool for secure file encryption using AES-256 combined with LSB steganography to embed encrypted payloads within carrier images — enabling covert, deniable data storage and transmission.

<div align="center">

| Attribute | Detail |
|:---|:---|
| **Stack** | Python · PyCryptodome · OpenCV · Tkinter · PIL · NumPy |
| **Scale** | Supports files up to 500MB · lossless embedding in PNG, BMP, TIFF formats |
| **Performance** | AES-256-GCM encryption in <1s for 10MB files · zero visual distortion (PSNR > 60dB) |
| **Security** | PBKDF2 key derivation · random IV per session · integrity-verified decryption |
| **Impact** | Demonstrated at college security symposium · adopted by 3 peer research groups |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-6d28d9?style=flat-square&logo=github&logoColor=white)](https://github.com/GopinathP7/cryptovault) |

</div>

CryptoVault separates encryption and steganography into independent, composable modules. The CLI supports scriptable pipelines for batch processing, while the GUI provides a drag-and-drop interface with live entropy visualisation of carrier images before and after embedding.

</details>
